# Use-case variants — router

The base move framework (`../move-framework.md`) describes the SOP genre in
general. Real statements differ by *what the applicant is applying for*. After
identifying the use case during intake, read the matching file below **in
addition to** the core files. Each file is a set of *changes* (deltas) layered
on top of the base — it does not repeat the base.

| Use case | File | Read it when… |
|---|---|---|
| Taught master's | `taught-masters.md` | A coursework-based MSc/MA; professional or academic, no major research component required. This is the closest to the base. |
| Research master's / PhD / doctoral | `research-phd.md` | An MRes, MPhil, PhD, or any research-driven program where fit with faculty and research potential dominate. |
| Career change | `career-change.md` | The applicant is pivoting fields. This is a **modifier** — layer it on top of `taught-masters.md` *or* `research-phd.md`, whichever fits the target. |
| Fellowship / scholarship / grant | `fellowship-scholarship.md` | A funded opportunity (national fellowship, research grant, scholarship) judged against published criteria. Can also layer on top of a degree use case. |

If more than one applies (e.g., a career-changer applying to a PhD, or a
master's applicant also writing a scholarship essay), read both relevant files;
the deltas compose.

## Sources behind these files

The differentiation rests on peer-reviewed genre analysis, primarily:

- Samraj, B., & Monk, L. (2008). *The statement of purpose in graduate program
  applications: Genre structure and disciplinary variation.* English for
  Specific Purposes, 27(2), 193–211. — the base move/step framework and the
  finding that disciplines share moves but differ in steps.
- Chiu, Y.-L. T. (2015). *Personal statement in PhD applications: Gatekeepers'
  evaluative perspectives.* Journal of English for Academic Purposes, 17, 63–73.
- Chiu, Y.-L. T. (2016). *'Singing your tune': Genre structure and writer
  identity in personal statements for doctoral applications.* Journal of English
  for Academic Purposes, 21, 48–59.
- López-Ferrero, C., & Bach, C. (2016). *Discourse analysis of statements of
  purpose: connecting academic and professional genres.* Discourse Studies,
  18(3), 286–310.
- Ganguly, P. (2020). *Rhetorical analysis of statements of purpose (SPs) for
  admission to graduate school.* Master's thesis, Missouri S&T.

Where a file draws on admissions-practitioner consensus rather than research
(e.g., MBA-essay conventions for career-changers), it says so explicitly. Do not
present practitioner guidance as a research finding.
